# AI Growth Planner – 6-Month Personal Development AI

This project uses:
- Next.js for Web Dashboard
- React Native for Mobile App
- NestJS for Backend
- Multi-Agent AI system (OpenAI) for planning, task generation, reporting

## AI Agents
1. SkillGapAgent
2. GoalPlannerAgent
3. DailyTaskAgent
4. ProgressTrackerAgent
5. HRReportAgent

## Goal
The system takes a user's role, level, skills and time availability → AI generates:
- 6-month OKRs
- weekly plan
- daily plan
- progress tracking
- monthly/6-month report

## Architectural vibe
- Modular NestJS backend
- Clean components for FE and App
- Server-driven UI (BE-first)
- API-first design
